Overview

In this notebook, we process the data from the online experiment studying Frame Induced Position Shift (FIPS).

Other resources for this project:

  1. OSF repository for the paper
  2. Preprint of the paper
  3. A demo version of the experiment

This Notebook is (generated from) an R markdown file that comes with an RStudio Project stored on GitHub. It includes scripts that will download the raw data and reproduce the figures and analyses presented in the paper. The data will, surprisingly, be stored in the folder called data, and the R scripts are in a folder called R.

Data

The data folder has a sub-folder called Pavlovia that has one csv file for every run of the online experiment that was recorded. Some participants have apparently done the task twice (or more), some may not have finished the task and others may not appear in the questionnaire data. It will be filled from the data.zip file on the OSF repository.

The experiment was part of a battery of tasks that consisted of 2 sessions, each of which had tasks accessible from a Qualtrics questionnaire. We used the first session to collect demographics information and the FIPS task was part of the second session, with some pre- and post task questions to ensure quality of the data. Therefore we need data from both questionnaires. We do not provide the raw data here as it contains unnecessary information that might break anonymity, and it has a lot of irrelevant information. Instead we use one cleaned csv file with only the necessary information. This has double and incomplete entries removed as well.

Effects of screen size and distance

For almost 50% of participants (69/141) we have reasonably reliable screen calibration data (viewing distance and pixel size in centimeters). First we look at a figure of the strength of the illusion (slope of a line fit to perceiced dot distance over frame movement amplitude in all 12 FIPS trials) as a function of how many degrees visual angle the frame subtends:

plotDVAeffects()

Looks like there is no effect of retinal frame size on the strength of the illusion. In order to confirm this we run a Bayesian regression:

DVAregression()
## Bayes factor analysis
## --------------
## [1] one_dva : 0.2542889 0%
## 
## Against denominator:
##   Intercept only 
## ---
## Bayes factor type: BFlinearModel, JZS

The Bayes Factor for the regression model is 0.254, which is moderate evidence for the null hypothesis (that retinal frame size does not affect illusion strength). So, we do not take this into account for any further analyses.

Effects of frame movement speed and period over amplitude

First, lets look at the data:

plotFIPSpercepts()

This figure uses 95% confidence intervals, whereas in the paper we chose to use SEM. The perceived distance between the dots seems pretty close to the distance the frame travels (dashed gray line), no matter if we keep velocity constant (and vary period) or if we keep period constant (and vary velocity).

To test this, we run a Linear Mixed Effects model, with participant as a random effect, condition (categorical: constant velocity or period) and frame movement (numeric) as fixed effects. We use REML and a Nelder-Mead optimizer:

LME <- FIPS_LME()
knitr::kable(coef(summary(LME)), digits=3)
Estimate Std. Error df t value Pr(>|t|)
(Intercept) 0.036 0.024 1200.583 1.484 0.138
framedist 0.910 0.037 1266.001 24.595 0.000
cond1 0.029 0.022 1266.001 1.364 0.173
framedist:cond1 -0.046 0.037 1266.001 -1.254 0.210

The distance the frame travels (framedist) predicts percept (t=24.6, p<.001) and has a coefficient of 0.91 which is getting pretty close to 1. There is no evidence that conditions contribute (t=1.4, p=.17) and they dont interact with frame distance either (t=1.3, p=.21). That is, in the model, the distance the frame moves is sufficient to explain the perceived distance between the dots.

Experiment 2: constant speed and constant period conditions

The online experiment was modelled after this task where the length of the path was varied while keeping either the speed constant or the period of movement. Path length seemed to be the only factor predicting perceived distance between the flashed dots, with no effect of condition (constant speed or constant period). Lets see if we can confirm the null hypothesis here:

lab_NwayAOV(file='data/experiment2_frame_movement.csv', within=c('path_length', 'condition'), printAOV = T)
## Anova Table (Type 3 tests)

## 3 path_length:condition 2.16, 15.09 0.48    6.83 ** .077    .007
## ---

## Sphericity correction method: GG

## [3] path_length:condition + participant                           : 0.1267069    0.6%
## [4] path_length + condition + participant                         : 8.807211e+34 1.64%
## [5] path_length + path_length:condition + participant             : 1.180794e+35 0.7%

## [7] path_length + condition + path_length:condition + participant : 1.169299e+35 1.67%
## 
## Against denominator:
##   percept ~ participant 
## ---
## Bayes factor type: BFlinearModel, JZS

Here the results of the two kind of statistical analyses are somewhat contradictory. The ANOVA here suggests that condition has an effect on perceived distance between dots (F(1,7)=14.6, p=.007) along with a main effect of path length (F(1.8,12.58)=120.92, p<.001, GG-corrected) and that there is an interaction between path length and condition as well (F(2.16,15.09)=6.83,p=.007, GG-corrected). However in the Bayesian statistics, there is extreme evidence for the main effect of path length (BF=9.8*e+34) but there is moderate evidence for a null effect of condition (BF=0.26) as well as for the interaction between path length and condition (BF=0.13). The model that explains most of the data is the model with only path length as predictor (accounting for effects of participant).
